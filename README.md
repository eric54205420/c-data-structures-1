# c-data-structures

Clone of https://github.com/jean553/data-structures in C.

## Requirements

### Ubuntu/Debian

```
apt-get install gcc cmake
```

### Archlinux

```
pacman -S gcc cmake
```

## Compilation

```
mkdir build && cd build/
cmake ..
make
```

## Execution

```
./bin/c-data-structures
```
