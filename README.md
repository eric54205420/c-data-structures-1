# c-data-structures

Clone of https://github.com/jean553/data-structures in C.

## Requirements

### Ubuntu/Debian

```bash
apt-get install gcc cmake
```

### Archlinux

```bash
pacman -S gcc cmake
```

## Compilation

```bash
mkdir build && cd build/
cmake ..
make
```

## Execution

```bash
./bin/c-data-structures
```
